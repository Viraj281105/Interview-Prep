# 🚀 Interview Prep Platform

> Master Technical Interviews with Structured Learning, Mock Interviews, and AI-Powered Feedback.

A modern, AI-powered interview preparation platform designed to help students and software engineering aspirants prepare smarter for technical interviews, placements, and internships.

This platform combines structured learning, interactive revision, mock interviews, progress tracking, and AI-driven feedback into one unified ecosystem.

---

# 🌟 Vision

Interview preparation is often messy and overwhelming.

Students usually switch between:

* scattered notes
* random YouTube videos
* coding platforms
* interview experiences
* last-minute revision sheets

This platform solves that problem by bringing everything into one place.

The goal is simple:

**Prepare smarter. Revise better. Perform confidently.**

# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/Viraj281105/Interview-Prep.git
```

Move into the project directory:

```bash
cd Interview-Prep
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
